# ТЗ 04 · План — «Інтерактивне планування»

## Мета й емоція
Дати раціональне підтвердження після емоції: показати, що 150 м² продумані. Користувач
наводить/тапає на кімнату — бачить площу й призначення. Відчуття «все обдумано».

## Патерн
Ядро: clickable/hoverable floor plan (SVG-план з інтерактивними зонами). Підтримка: count-up площі.

## Ассети
- План будинку (з `Планування_Нагірна.pdf` → перемалювати у чистий SVG або підготовлений PNG-підклад).
- Дані кімнат (точні, з CLAUDE.md):
  - Майстер-спальня 16.65 м² (ensuite + гардероб)
  - Спальня 14.43 м², Спальня 13.41 м²
  - Кухня-вітальня 45 м², Тераса 29.83 м²
  - Центральний гардероб + санвузол, Навіс для авто 36.92 м²

## Поведінка / взаємодія
- Hover/tap на зону → підсвітка + tooltip/панель з назвою і площею.
- Опційно: загальний лічильник «150 м²» count-up при вході.
- Активна зона — теплий акцент (`--color-accent`), решта приглушена.

## Текст (з content/COPY_nahirna.md)
- Заголовок секції про продумане планування.
- Назви кімнат + площі (точні, не змінювати).

## Технічні обмеження
- SVG-зони з aria-label (доступність), keyboard-навігація (tab + enter).
- reduced-motion → без анімації підсвітки, миттєвий стан.
- mobile: tap замість hover, панель площі знизу.

## Definition of Done
- [ ] усі зони клікабельні, площі точні
- [ ] keyboard + aria доступність
- [ ] mobile tap працює
- [ ] reduced-motion fallback
