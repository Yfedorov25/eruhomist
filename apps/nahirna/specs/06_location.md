# ТЗ 06 · Локація — «Вінниця, вул. Нагірна»

## Мета й емоція
Показати, що дім — і усамітнений (глухий кут, берег), і зручний (близько до міста/інфраструктури).
Зняти заперечення «чи не далеко?». Баланс приватності й доступності.

## Патерн
Ядро: мапа з точками інтересу (POI). Підтримка: reveal списку відстаней.

## Ассети
- Мапа (статичний стилізований кадр або легка інтерактивна — без важких SDK, якщо б'є по LCP).
- Маркер дому на вул. Нагірна + POI.
- ⚠️ Точні відстані до POI — УТОЧНИТИ В КЛІЄНТА (зараз плейсхолдери). Не вигадувати числа.

## Поведінка / взаємодія
- Маркери POI з підписами (центр міста, школа/садок, магазини, виїзд на трасу — за даними клієнта).
- При вході — stagger-reveal списку відстаней.
- Hover на маркер → підсвітка + назва.

## Текст (з content/COPY_nahirna.md)
- Теза про баланс «тиша + близько до міста».
- Список POI з відстанями (вставити після уточнення клієнтом).

## Технічні обмеження
- Якщо інтерактивна мапа — lazy-load, не блокувати LCP. Інакше — статичний стилізований кадр.
- reduced-motion → список без stagger.
- mobile: вертикальний список + компактна мапа.

## Definition of Done
- [ ] мапа з маркерами, не б'є по LCP (lazy)
- [ ] відстані — реальні (після клієнта), не плейсхолдери у фіналі
- [ ] mobile + reduced-motion fallback
