# ТЗ 02 · Архітектура — «Крізь скло»

## Мета й емоція
Показати якість архітектури й матеріалів (клінкер, панорамне скління, дах) через прийом
«погляд крізь скло»: зовні → всередину. Відчуття добротності, теплоти, «зроблено надовго».

## Патерн
Ядро: sticky window-reveal (маска/кадрування, що розкриває інтер'єр крізь віконний отвір на scroll).
Підтримка: stagger-reveal деталей-підписів (матеріали).

## Ассети
- Екстер'єр день (один з реальних рендерів 11/12/13/14/18/21/22/25 — обрати фронтальний).
- Інтер'єр день (A1 — кухня-вітальня) для «крізь скло».
- Деталі матеріалів (опційно, кроп з рендерів): цегла, дах, колона з білим капітелем.

## Поведінка скролу
- Секція sticky на ~150vh.
- На scroll «вікно» (маска) розкривається: зовнішній план поступається інтер'єру крізь скління.
- Підписи-деталі fade-in поетапно: «клінкерна цегла», «панорамне скління», «глибокі звиси даху».

## Текст (з content/COPY_nahirna.md)
- Теза про якість будівництва й матеріали (натуральні, надовго).
- 2–3 короткі підписи-деталі.

## Технічні обмеження
- Маска через clip-path / CSS mask на scrub (GPU-friendly).
- reduced-motion → статичні два кадри поруч (зовні + всередині), без маски.
- mobile: спрощений reveal (fade замість складної маски).
