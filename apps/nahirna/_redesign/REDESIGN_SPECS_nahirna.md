# 🅰 СИСТЕМА ЧИТАБЕЛЬНОСТІ (база для ВСІХ секцій — зробити першою)

## Принцип: три рівні захисту
- **Рівень 1 · `Scrim`** — градієнтний overlay (НЕ суцільна плашка) між медіа і текстом.
  Параметри: `direction` (top|bottom|left|right), `strength` (0–1 множник альфи).
  `pointer-events:none`, `position:absolute inset:0`. Напрямки:
  - top: `linear-gradient(to bottom, rgba(15,15,14,.45) 0%, transparent 120px)`
  - bottom: `linear-gradient(to top, rgba(15,15,14,.78) 0%, transparent 45%)`
  - left: `linear-gradient(to right, rgba(15,15,14,.7) 0%, transparent 55%)`
- **Рівень 2 · `TextShelf`** — диптих: текст живе на суцільній шоколадній панелі (`#1a1510`/`#100c08`),
  фото поруч (не під ним). Нечитабельність неможлива. Використати на 02.
- **Рівень 3 · safe-zone** — правило вибору кадру: текст на ТЕМНУ частину кадру; якщо світла → TextShelf.

## Застосування по екранах
- Hero надзаголовок: `Scrim top strength=0.5` + текст `#EDEAE3` (НЕ золото).
- Hero H1: `Scrim bottom strength=0.6` під блоком.
- 02: повністю на TextShelf.
- 06: фон темний — додатковий scrim не потрібен.
- CallPill: не чіпати, читається завжди.

## DoD

# 🅱 ТЗ 02 · АРХІТЕКТУРА — «Зроблено для тих, що залишаються» (ПЕРЕРОБКА)

## Прибрати
Старий «window-reveal крізь скло» (інтер'єр виїжджає з-за фасаду) — ВИДАЛИТИ повністю. Читається як глюк.

## Патерн
Ядро: **text-shelf диптих** — текст-полиця ліворуч (шоколад) + кадр-вікно праворуч.
У кадрі-вікні — **day→night dissolve на scroll-scrub** (НЕ мош, а розчинення: день fade-out,
ніч fade-in, + scale 1.0→1.04 для «дихання»). Знизу — тихий рядок 3 деталей.

## Layout (desktop)
```
[ПОЛИЦЯ #1a1510 ~38%]   [КАДР-ВІКНО ~62%]
 надрядок: матеріали      ┌─ day ─┐
 H2 (день/ніч стан)       │ dissolve │
 абзац про матеріали      └─ night ┘
 місток (курсив)
─────────────────────────────────────
[Клінкер]  [Колона]  [Дах]  ← тихі тези
```

## Ассети (звір у /public/images/)
- `exterior-terrace-columns.webp` — кадр-вікно, стан «день» (тераса/колони крупно).
- `exterior-night-1.webp` (або `-2`) — кадр-вікно, стан «ніч» (фасад з підсвіткою).
- ⚠️ День і ніч різного ракурсу → НЕ піксельний crossfade-мош, а **dissolve** (opacity + scale).
  Невеликий зсув ракурсу читається як «той самий дім в інший час», не як глюк. Навмисне.
- Якщо нічний зовсім інший за композицією — обрати найближчий за ракурсом.

## Поведінка скролу
- pin ~150vh. 0→100%: day opacity 1→0, night 0→1 (dissolve, scrub); обидва scale 1.0→1.04.
- Полиця: H2 «день» на старті; на ~50% мʼяко fade-through на стан «ніч»; не різко.
- Рядок деталей: stagger fade-in коли текст у центрі viewport (по черзі, 1.5s power3.out).
- Текст полиці завжди читабельний (шоколад) — scrim не потрібен.

## Текст (з COPY_REBUILD §02 — дослівно)
- Надрядок: `матеріали`
- H2 день: `Удень цей дім — про камінь.` · H2 ніч: `А коли темніє — про світло.`
- Абзац: `Темна клінкерна цегла й білий камінь на колонах. Такі матеріали не оновлюють кожні
  пʼять років — вони просто гарнішають. Цей дім будували для тих, що залишаються.`
- Місток (курсив): `Один поверх. Між Вами і ранковою кавою на терасі — жодної сходинки.`
- Деталі (3, повні): Клінкер / Колона / Дах — дослівно з COPY_REBUILD.

## Технічні обмеження
- Dissolve через opacity двох absolute-шарів + scale (тільки transform/opacity).
- reduced-motion → один кадр (нічний, драматичніший) + повний текст полиці, без dissolve.
- mobile: полиця зверху, кадр під нею (стек); dissolve легкий; pin короткий або без pin.
- next/image, lazy, WebP.

## DoD

- [ ] Dissolve день→ніч плавний, без «наїжджання».
- [ ] Текст полиці читабельний завжди (Playwright).
- [ ] Деталі знизу — повні тези, не обрізані.
- [ ] reduced-motion + mobile стек. GSAP cleanup.

# 🅲 ТЗ 06 · ЛОКАЦІЯ — «Здається, що за містом» (НОВА КАРТА)

## Прибрати
Намальовану SVG-схему (коричнева дорога-крива + синя пляма). Виглядає дешево.

## Патерн
**Статичний супутниковий кадр** (НЕ живий embed — нуль навантаження на швидкість), оброблений
під нашу палітру, + кінематографічні **scroll-driven шари** зверху. Права колонка з цифрою.

## Ассети
- `map-satellite.webp` — статичний супутниковий/аерофото кадр Вінниці з вигином Бугу.
  ⚠️ TODO власнику (джерело+ліцензія): Google/Bing static, або відкриті тайли (OSM/Sentinel).
  Підтвердити право на комерційне використання ПЕРЕД релізом.
  - Дім середній: видно місто + вигин ріки + район. Пін на РАЙОНІ, не на конкретному дворі
    (приватність: не розкриваємо точну адресу публічно).
- Обробка (CSS-filter або заздалегідь у WebP): затемнити, теплий тон, наша палітра (шоколад-ніч).
  Не лишати «гугл-сірим».

## Шари зверху (SVG/DOM overlay, на скрол по черзі)
1. **Золотий пін** `#E8C9A0` з пульсівним ореолом — на районі Нагірної.
2. **Контур ділянки** — тонка золота лінія (тільки якщо не видає точну адресу; інакше пропустити).
3. **Лінія «власного берега»** — пунктирна золота вздовж води.
4. **Тихі підписи:** `ВІННИЦЯ` (вгорі), `ПІВДЕННИЙ БУГ` (внизу), letter-spacing, дрібно.
- Порядок появи: пін → (контур) → лінія берега → підписи. Кожен fade-in + легкий рух, 1.2–1.8s, stagger.

## Права колонка (текст — з COPY_REBUILD §06, дослівно)

- ⚠️ TODO власнику: реальні хвилини → замінити на 3 POI (центр/школа/садок).

## Технічні обмеження

# 🅳 ТЗ ЛОАДЕР · «Рівень води» (НОВИЙ)

## Прибрати
Плоский текстовий лоадер. Замінити на бренд-момент.

## Патерн
Темна вода піднімається 0→100% знизу вгору (CSS clip-path/height + легка SVG-хвиля). На 100% —
«розлив» вгору (curtain-reveal), що відкриває hero. Маскує preload перших ~20 кадрів hero.

## Поведінка
- Фон `#0F0F0E`. Вода — `#16323d` з тонкою лінією поверхні `#3a6575`.
- Рівень 0→100% за ~1.8–2.5s, power2.inOut (luxury).
- Поверхня — легка SVG-хвиля (1–2 синуси, дуже повільно, transform only).
- Слово над поверхнею (тихо, тепло-біле): `Власний берег`.
- Лічильник (тонко, золото): `0 → 100`.
- На 100%: вода «розливається» вгору 0.8s power4.out → перехід у hero (зʼявляється hero-day).
- НЕ галʼмувати штучно: якщо ассети готові раніше — мінімум 1.2s показу, потім reveal. Timeout-fallback.

## Текст (з COPY_REBUILD)
- `Власний берег` (над водою). Жодного «Завантаження».

## Технічні обмеження
- Тільки transform/opacity/clip-path. Без важких бібліотек.
- reduced-motion → пропустити анімацію, одразу статичний hero-day (без води).
- НЕ блокувати LCP: hero-day має вантажитись паралельно; лоадер маскує саме preload кадрів.
- mobile: те саме, легша хвиля.

## DoD

- [ ] На 100% розлив угору відкриває hero без стрибка (CLS=0).
- [ ] «Власний берег» читабельний над водою.
