# BRIEF — секції сайту Єрухомість (для Claude Code)

> Скопіюй цей файл у Claude Code. Виконуй по черзі, секція за секцією.
> Перед стартом: прочитай `CLAUDE.md` і `content/eruhomist-data.json`.

---

## 0. Підготовка

```
1. Поклади content/eruhomist-data.json у <project>/content/eruhomist-data.json
2. Поклади download-assets.mjs у корінь проєкту, запусти:  node download-assets.mjs
   -> фото об'єктів з'являться в public/objects/
   Якщо якісь URL віддали 403 (протермінувались) — відкрий .post-посилання
   зі скрипта і збережи фото вручну в public/objects/ з тим самим іменем.
3. Опційно: сконверти .jpg -> .webp через sharp для меншої ваги.
```

Усі секції імпортують дані з `content/eruhomist-data.json`. Не хардкодь ціни/площі —
бери з JSON, щоб легко оновлювати.

---

## Спільні правила (для всіх секцій)

- Естетика з `CLAUDE.md`: фон `#0a0c0f`, акцент `#e8a35c`, білий текст, тонкі
  великі заголовки (font-weight 300), акцентні слова — 600.
- `"use client"` на секціях з моушеном. Моушен — GSAP + ScrollTrigger.
- Reveal-патерн: на in-view елемент виїжджає знизу (y:40→0, opacity 0→1),
  внутрішні елементи зі stagger 0.1–0.15s. Повільно, ease "power3.out".
- Мова — українська. Тон бренду: впевнений, преміальний, людяний.
- Кожну секцію після створення підключай у `app/page.js` у правильному порядку.
- Зображення — через next/image, з плейсхолдером поки фото немає.

Фінальний порядок у `app/page.js`:
```
<HeroSection /> <Directions /> <Catalog /> <Investment /> <Cases /> <About /> <Contact />
```

## Секція 02 — Directions (Три напрямки)

```
Створи components/Directions.jsx. Дані: directions[] з eruhomist-data.json.
3 повноширинні блоки один під одним, кожен ~80vh, чергуються ліво/право
(block 01 текст зліва, 02 справа, 03 зліва).
Кожен блок: великий приглушений номер (01/02/03, напівпрозорий #e8a35c, ~120px),
тонкий заголовок, абзац (rgba(255,255,255,.7), max-width 440px),
зображення на протилежній стороні (next/image, з directions[].image).
Моушен: на in-view — заголовок виїжджає знизу, номер+текст зі stagger.
Розділювач між блоками — тонка лінія rgba(255,255,255,.08).
```

## Секція 03 — Catalog (Каталог об'єктів)

```
Створи components/Catalog.jsx. Дані: objects[] з eruhomist-data.json.
Заголовок секції зліва ("Об'єкти" + підзаголовок).
Сітка карток (на десктопі 3 в ряд, планшет 2, моб 1).
Картка об'єкта:
  - фото зверху (next/image, objects[].image, співвідношення 4:3, hover: лёгкий zoom)
  - бейдж типу (objects[].type) і статусу (objects[].status) поверх фото
  - заголовок (objects[].title)
  - локація (objects[].location) з іконкою
  - рядок: площа (area) + ціна (price), ціна акцентним #e8a35c
  - 2-3 теги (objects[].tags) дрібним шрифтом
  - hover: тонка рамка #e8a35c, плавно
Моушен: картки з'являються зі stagger при скролі.
Знизу — CTA "Дивитись усі об'єкти" або "Підібрати під запит" (-> #contact).
ВАЖЛИВО: ціни/площі брати рівно з JSON, не змінювати.
```

## Секція 04 — Investment (Інвестиції та девелопмент)

```
Створи components/Investment.jsx. Дані: investment{} з eruhomist-data.json.
Темна секція, акцент на цифрах.
Великий заголовок ("Капітал, що працює" або подібне в тоні бренду).
3-4 метрики великим шрифтом: поріг входу (investment.entry = "від 20 000 $"),
дохідність (investment.yield = "до 25% річних"), + ще пара.
Блок "4 фактори прибутковості" (investment.factors) — нумерований список або
горизонтальний ряд.
Формати (investment.formats) — пілюлі/теги.
Моушен: цифри counter-up або fade-in зі stagger при in-view.
CTA: "Дізнатись про інвестиції" (-> #contact).
```

## Секція 05 — Cases (Кейси / результати)

```
Створи components/Cases.jsx. Дані: cases[] з eruhomist-data.json.
Заголовок ("Результати" / "Кейси").
3 кейси — великі блоки або картки. Кожен:
  - велика метрика (cases[].metric: "13/13", "2 об'єкти", "100% онлайн") акцентом
  - заголовок (cases[].title)
  - текст (cases[].text)
Подача в стилі "доказів" — впевнено, цифри на першому плані.
Моушен: reveal зі stagger, метрика може мати counter-анімацію.
```

## Секція 06 — About (Про нас + таймлайн)

```
Створи components/About.jsx. Дані: timeline[] + contact.values з eruhomist-data.json.
Дві частини:
1. Короткий маніфест бренду (2-3 речення в тоні: "Єрухомість — про людей,
   ідеї та масштабні можливості. Від перших кроків до девелоперських проєктів").
   + цінності (contact.values) як теги.
2. Таймлайн 2022->2025 (timeline[]): вертикальний або горизонтальний,
   рік акцентом #e8a35c, поряд текст. Лінія-з'єднувач.
Фото команди: public/objects/brand-team.jpg (next/image).
Моушен: пункти таймлайну з'являються один за одним при скролі (stagger).
```

## Секція 07 — Contact (Контакт / CTA)

```
Створи components/Contact.jsx з id="contact". Дані: contact{} з eruhomist-data.json.
Велика фінальна секція, темна.
Заголовок-заклик ("Знайдемо те, що працює саме для вас").
Контакти: телефон (contact.phone) як tel: лінк, Instagram (contact.instagram).
Проста форма: ім'я, телефон/контакт, повідомлення (поки без бекенду —
console.log або mailto, познач TODO: підключити надсилання).
Кнопка submit акцентом #e8a35c.
Внизу — місто (Вінниця), @e_ruhomist.
Моушен: fade-in секції.
ВАЖЛИВО: форму НЕ підключай до жодного зовнішнього сервісу без окремої вказівки.
```
