Контекст проєкту для Claude Code. Читай цей файл перед будь-якою задачею в репозиторії.

## Що це за проєкт

Сайт агенції нерухомості **«Єрухомість»** (Вінниця). Кінематографічний, темний,
преміальний сайт із scroll-controlled hero (камера заходить ззовні будинку всередину
під час скролу). Естетика рівня Igloo / Lusion — преміум-моушн, не «AI-vibe-coded».

Hero вже згенеровано за scroll-craft pipeline (Kling → FFmpeg → canvas → GSAP).
Решта секцій будується навколо нього в тій самій естетиці.

## Естетика / дизайн-система

- **Тема:** темна, кінематографічна.
- **Фон:** `#0a0c0f` (майже чорний).
- **Акцент:** `#e8a35c` (тепле золото — взято з кольору світла в hero-кадрах).
- **Текст:** білий / `rgba(255,255,255,.75)` для вторинного.
- **Типографіка:** тонкі великі заголовки (font-weight 300) з акцентними словами
  жирним (600). Великі розміри (clamp). Letter-spacing на kicker-ах (uppercase,
  .3–.45em). Уникай Inter/Roboto/Arial — обери характерний display-шрифт.
- **Композиція:** багато негативного простору, текст у лівій третині кадру,
  vignette, асиметрія. Преміум = повітря + контроль, не щільність.
- **Рух:** плавний, повільний, «дорогий». Stagger-револи на load. Scroll-тригери.
  GSAP для всього моушену.
- **Мова інтерфейсу:** українська.

## Технічний стек

- **Next.js** (App Router), React.
- **GSAP + ScrollTrigger** для скролу й моушену (`npm i gsap`).
- Кадри hero — webp у `/public/hero/desktop` і `/public/hero/mobile`.
- Клієнтські компоненти з моушеном помічай `"use client"`.

## Структура сайту (секції)

1. **Hero** — scroll exterior→interior (✅ готово, `components/HeroSection.jsx`)
2. **Три напрямки** — підбір / забудовники / інвестиції
3. **Каталог об'єктів** — картки (ціна, площа, локація, тип)
4. **Девелопмент / інвестиції** — поріг входу, дохідність, як працює
5. **Кейси / результати** — «13 таунхаусів за 5 міс», «50k$ → будинок + інвестоб'єкт»
6. **Про нас + таймлайн** — шлях 2022→2025, команда
7. **Контакт / CTA** — Direct, телефон, форма заявки

## Конвенції коду

- Компоненти — у `components/`, секції сайту — окремими файлами (`Directions.jsx`,
