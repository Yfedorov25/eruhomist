## 0. СУТЬ ПРОЄКТУ
QUADRO HOUSE — клубний дім на **4 квартири**, Вінниця (Корея), вид на озеро, 7 хв до центру.
Головна ідея: єдиний у місті дім, де лише чотири квартири — приватність без заміського клопоту.
Драматургія сайту = **один день будинку: ранок → вечір**. Скрол керує переходом дня в ніч.
Двомовність UK/EN (i18n, перемикач). Деплой Vercel.

## 1. ГЛОБАЛЬНА ДРАМАТУРГІЯ (наскрізний прийом)
Hero = day→night frame-sequence (готові відео нарізаються на кадри).
Тема сторінки повертається зі **світлої** (день, верх сайту) у **темну** (вечір, низ) по мірі скролу.
Один persistent canvas для hero; секції нижче — звичайний DOM + GSAP reveal поверх теми.

## 2. КАРТА СЕКЦІЙ

### S1 — HERO (recipe R01 + R05)
- **Медіа:** `public/videos/hero-desktop.mp4` (16:9, 4K, 6s, 145 кадрів — фінал 48370) →
  нарізати у `public/frames/hero/desktop/0001..0145.webp` (1920w).
  Мобільне: `hero-mobile.mp4` (9:16, 6s, 145 кадрів — фінал 29197) → `frames/hero/mobile/` (1080w).
- **Поведінка:** scroll-scrubbed frame-sequence у WebGL (R01-скелет). Скрол = індекс кадру = час доби.
  Камера в обох відео нерухома. ShaderMaterial: film grain 0.04 + vignette 0.5 (desktop), grain off на mobile.
- **ScrollControls:** pages=4, damping=0.15. Перші 20 кадрів eager preload, решта progressive.
- **Текст поверх (з messages, НЕ хардкод):**
  - H1: «У цьому домі буде лише чотири квартири.»
  - Sub: «Вид на озеро, сім хвилин до центру і троє сусідів замість трьохсот.»
  - Заголовок проявляється повільно (SplitText, 1.5–2s) на початку скролу, м'яко зникає до переходу в ніч.
- **Fallback:** prefers-reduced-motion → статичний кадр 0001 (день) + текст, без scrub.
  Mobile-low → `<picture>` день/ніч замість WebGL, якщо fps < 30.

## 3. ТИПОГРАФІКА / ПАЛІТРА
- Display: distinctive (контрастна антиква АБО high-contrast grotesk; НЕ Inter/Roboto/Space Grotesk).
- Body: чистий рефінований гротеск.
- CSS variables: `--bg`, `--fg`, `--accent` (тепле світло вікон ~2700K), `--deep` (вечірній синій).
- Hero керує темою: світла зверху → темна знизу (через scroll progress на :root).
- Grain/vignette лише на hero. Інші секції — чисті, з глибокими тінями вечора.

## 4. PERFORMANCE / QA
- LCP < 2.5s, CLS < 0.1, 60fps desktop / ≥40fps mobile.
- Hero WebGL desktop-first; mobile fallback за потреби.
- Lazy-load усе below-the-fold; preload лише перші 20 кадрів hero.
- QA: Playwright скрол-скріни на 0% / 25% / 50% / 75% / 100% — звірити день→ніч і читабельність тексту.
- design-review (anti-slop) кожної секції: оцінка 0–10, нижче 8 — переробити.
